# Rest Api - Nodejs

Práctica de mi primera rest api con node js

### Pre-requisitos 📋

-   Instalar node

### Instalación 🔧

-   Clona este proyecto y luego entra a la carpeta y ejecuta el comando

```
npm install
```

Y luego

```
npm start
```

De esta manera tendrás la api funcionando en el puerto 3000
